Intro

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns,
making it easier to design web pages without having to use floats and positioning.

Grid Elements

A grid layout consists of a parent element, with one or more child elements.
<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
</div>

Display Property

Set its display property to grid or inline-grid.

  • .grid-container { display: grid; }
  • .grid-container { display: inline-grid; }

Grid Gaps

grid-column-gap; grid-row-gap; grid-gap; (Adjust the gap size)
   .grid-container { display: grid; grid-row-gap: 50px; }

Grid Lines

  • Column lines: lines between columns.
  • Row lines: lines between rows.
Place a grid item at column line 1, and let it end on column line 3:
   .item1 { grid-column-start: 1; grid-column-end: 3;}

Grid Container

grid-template-columns

grid-template-rows: defines numbers of columns & width of each column.

If you want your grid layout to contain 4 columns, specify the width of the 4 columns, or "auto" if all columns should have the same width:
   .grid-container { display: grid; grid-template-columns: 80px 200px auto; }

grid-template-rows

Same as grid-template-columns.

justify-content

Align the whole grid inside the container.
** The grid's total width has to be less than the container's width for the justify-content property to have any effect.

  • space-evenly: give the columns equal amount of space between, and around them.
  • space-around: give the columns equal amount of space around them.
  • space-between: give the columns equal amount of space between them.
  • center: align the grid in the middle of the container.
  • start: align the grid at the beginning of the container.
  • end: align the grid at the end of the container.

align-content

Vertically align the whole grid inside the container.
** The grid's total height has to be less than the container's height for the align-content property to have any effect.

  • space-evenly: give the rows equal amount of space between, and around them.
  • space-around: give the rows equal amount of space around them.
  • space-between: give the rows equal amount of space between them.
  • center: align the rows in the middle of the container.
  • start: align the rows at the beginning of the container.
  • end: align the rows at the end of the container.

Grid Item

By default, a container has one grid item for each column, in each row.

grid-column

Shorthand property for the grid-column-start and the grid-column-end properties.

  • Start on row-line 1 and end on row-line 4:
       .item2 { grid-column: 1 / 5; }
  • Start on row 1 and span 2 rows:
       .item2 { grid-column: 1 / span 2; }

grid-row

Same as grid-column.

grid-area

Shorthand property for the grid-row-start, grid-column-start, grid-row-end and the grid-column-end properties.

  • Make "item8" start on row-line 1 and column-line 2, and end on row-line 5 and column line 6:
       .item3 { grid-area: 1 / 2 / 5 / 6; }
  • Make "item8" start on row-line 2 and column-line 1, and span 2 rows and 3 columns:
       .item3 { grid-area: 2 / 1 / span 2 / span 3; }

Naming Grid Items

  • Item1 gets the name:
       .item1 { grid-area: myArea; }
  • Define 2 rows with 5 columns items names:
       .grid-container { grid-template-areas: 'myArea myArea . . .' 'myArea . . .';}

Order of the Items

The first item does not have to appear as the first item in the grid.
.item1 { grid-area: 1 / span 3 / 2 / 4; }
.item2 { grid-area: 3 / 3 / 4 / 4; }
.item3 { grid-area: 2 / 1 / 3 / 2; }
.item4 { grid-area: 2 / 2 / span 2 / 3; }
.item5 { grid-area: 3 / 1 / 4 / 2; }
.item6 { grid-area: 2 / 3 / 3 / 4; }
