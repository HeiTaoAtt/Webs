Block Elements Align

块元素指的是占据全部可用宽度的元素，并且在其前后都会换行。

margin

可通过将左和右外边距设置为 "auto"，来对齐块元素。
如果宽度是 100%，则对齐没有效果。
.center { margin-left:auto; margin-right:auto; width: 70%; }

position

绝对定位.
.right { position: absolute; right: 0px;}

float

.right { float: right;}

Dimension

控制元素的高度和宽度; 增加行间距。
height; line-height; max-height; max-width; min-height; width;

Classification 分类

display

设置为内联/外联元素: p { display: inline; }
div { display: block; }

clear

图像的左侧和右侧均不允许出现浮动元素:
img {float: left; clear: both;}

cursor

改变光标样式:
default; auto; crosshair; pointer; move; text; wait; help; etc.

visibility

使元素可见/不可见:
h2 { visibility: hidden; }
h2 { visibility: visible; }

Navigation Bar

  1. 导航栏基本上是一个链接列表，因此使用 <ul> 和 <li> 元素是非常合适的.
  2. 从ul列表中去掉圆点(list-style-type: none); 外边距(margin: 0; padding: 0;)

Vertical Navigation

在以上的基础上，定义 <a> 元素的样式 a { display: block; width: 60px; }

  • display:block - 把链接显示为块元素可使整个链接区域可点击（不仅仅是文本），同时也允许我们规定宽度。
  • width:60px - 块元素默认占用全部可用宽度，链接无法滑动至彼此相邻。我们必须规定块元素宽度。

Horizontal Navigation

使用行内列表项 / 浮动列表项.
*如果希望链接拥有相同的尺寸，就必须使用浮动方法。

  1. 行内列表项

    在以上的基础上，定义 <li> 为行内元素。
    li {display: inline;}
    • 这样，我们去除了每个列表项前后的换行，以便让它们在一行中显示。

  2. 浮动列表项

    在以上的基础上，浮动 <li> 元素并规定 元素的宽度。
    li { float: left; }
    a { display: block; width: 60px; }
    • float:left - 使用 float 来把块元素滑向彼此
    • display:block - 把链接显示为块元素可使整个链接区域可点击（不仅仅是文本），同时也允许我们规定宽度。
    • width:60px - 由于块元素默认占用全部可用宽度，链接无法滑动至彼此相邻。我们必须规定块元素宽度。

Opacity 图像透明度

img { opacity: 0.4; filter: alpha(opacity=40);}
//opacity value 0~1

  1. hover

    当鼠标指针移到图像后，图像会不透明。
    img { opacity: 0.4; filter: alpha(opacity=40);}
    img:hover {opacity: 1.0; filter: alpha(opacity=100);}

  2. 透明框中的文本

    <div class="background"></div>
    **div 有固定的高度和宽度, 背景图像, 边框.
    <div class="transbox"> This is some text in transparent box. </div>
    **稍小的div, 有固定的宽度, 背景色, 边框; 透明.

Media Types

定义以何种媒介来提交文档。

Media Query

Include a block of CSS properties only if a certain condition is true.

@media only screen and (max-width: 600px)
{
body { background-color: lightblue; }
}
**If the browser window is 600px or smaller, the background color will be lightblue.

Breakpoint

Makes the screen look good on a small screen.

@media only screen and (max-width: 768px) {
[class*="col-"] {
width: 100%;
}}
When the screen (browser window) gets smaller than 768px, each column should have a width of 100%.

Always Design for Mobile First

Instead of changing styles when the width gets smaller than 768px,
we should change the design when the width gets larger than 768px.

Hide Elements

Hide elements on different screen sizes.

@media only screen and (max-width: 600px) {
div.example {
display: none;
}}
