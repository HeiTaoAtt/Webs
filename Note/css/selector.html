Element Selector

Select every element based on element name.

eg. head, div, p, h1, etc.

Attribute Selector

Select every element with a spacific attribute and value.

eg. title, href, name, etc.

  1. <p class="is important">

    This paragraph is selected.

  2. <p class="warning">

    This paragraph is not selected.

[abc^="def"]: select abc attribute has value start with "def".
[abc$="def"]: select abc attribute has value end with "def".
[abc*="def"]: select abc attribute has value contains "def".
[abc~="def"]: select abc attribute has word value contains "def".

Derived Selector

CSS:
"h2 strong {color: blue; }"

HTML:

  • "<p>The strongly emphasized word in this paragraphis<strong>red</strong>.</p>"
  • "<h2>This subhead is also red.</h2>"
  • "<h2>The strongly emphasized word in this subheadis<strong>blue</strong>.</h2>"
Only last sentence will be effected.

ID Selector

CSS:
"div#sidebar { border: 1px dotted #000; padding: 10px; }"

HTML:
"<div id="sidebar">Effect this div content.</div>"

Class Selector

CSS:
"h1.center {text-align: center}"

HTML:

  • "<h1 class="center"> This heading will be center-aligned </h1>"
  • "<p class="center"> This paragraph will not be center-aligned.</p>"

Descendant Selector 后代选择器

CSS:
h1 em {color: red;}

HTML:

  1. <h1>This will <em>be effected </em> !</h1>
  2. <p>This will <em>not be effected </em> !</p>

Child Selector 子元素选择器

CSS:
h1 > strong {color:red;}

HTML:

  1. <h1>This is <strong>very</strong> <strong>very</strong> important.</h1>
  2. <h1>This is <em>really <strong>very</strong></em> important.</h1>
Two elements in 1st will be effected, no element in 2nd will be effected.

Adjacent Sibling Selector 相邻兄弟选择器

CSS:
h1 + p {margin-top:50px;}
选择紧接在 h1 元素后出现的所有段落，h1 和 p 元素拥有共同的父元素

Pseudo-classes

Anchor pseudo class

selector : pseudo-class {property: value}
eg.
  • a:visited{color: red;}
  • a.red : visited {color: #FF0000}

First-child pseudo class

元素的第一个子元素.
p:first-child {font-weight: bold;}

Psuedo-elements

selector:pseudo-element {property:value;}

:first-line

向文本的首行设置特殊样式.
只能用于块级元素. p:first-line { color:#ff0000; }

:first-letter

向文本的首字母设置特殊样式.
只能用于块级元素. p:first-letter { color:#ff0000; }

:before

在元素的内容前面插入新内容。 h1:before {content:url(logo.gif);}

:after

在元素的内容后面插入新内容。 h1:after {content:url(logo.gif);}
