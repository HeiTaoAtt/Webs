Flexbox Elements

Define a flex container with three flex items:
<div class="flex-container"> <div>1</div> <div>2</div> <div>3</div> </div>

Parent Element (Container)

The flex container becomes flexible by setting the display property toflex:
   .flex-container {display: flex;}

flex-direction

Defines in which direction the container wants to stack the flex items. row; column;
   .flex-container { display: flex; flex-direction: column-reverse; }

flex-wrap

Specifies whether the flex items should wrap or not. nowrap; wrap; wrap-reverse;
   .flex-container {display: flex; flex-wrap: wrap;}

flex-flow

Shorthand property for setting both the flex-direction and flex-wrap properties.
   .flex-container {display: flex; flex-flow: row wrap;}

justify-content

Aligns flex items of the container.
center; flex-start(Default); flex-end;
space-around(space between flex items); space-between(evenly space before, after each flex item);
   .flex-container {display: flex; justify-content: center;}

align-items

Align the flex items vertically.
center; flex-start; flex-end;
stretch(Default)(stretch to fill the container); baseline(based on first flex item baseline);

align-content

With the flex-wrap property set to wrap.
center; flex-start; flex-end;
space-around; space-between;
stretch(Default)(stretch flex lines to fill remaining space);
   .flex-container { display: flex; flex-wrap: wrap; align-content: space-between;}

Perfect Centering

Set both the justify-content and align-items properties to center.
   .flex-container {display: flex; justify-content: center; align-items: center; }

Child Element (Items)

order

<div style="order: 3"> 1 </div>
<div style="order: 1"> 1 </div>
<div style="order: 2"> 1 </div>

flex-grow

Specifies how much a flex item will grow relative to the rest of the flex items.
<div style="flex-grow: 1"> 1 </div>
<div style="flex-grow: 8"> 2 </div>
**Let item2 be 8 times bigger than item1.

flex-shrink

Specifies how much a flex item will shrink relative to the rest of the flex items.
<div style="flex-shrink: 0"> 3 </div>
<div> 4 </div>
**Do not let the third flex item shrink as much as the other flex items.

flex-basis

Specifies the initial length of a flex item.
<div style="flex-basis: 200px">3</div>

flex

Shorthand property for the flex-grow, flex-shrink, and flex-basis properties.
<div style="flex: 0 0 200px"> 3 </div>

align-self

Overrides the default alignment set by the container's align-items property.
<div style="align-self: center"> 3 </div>
